# Marketing Spend vs Revenue Prediction using Least Squares Estimation

## Project Overview
This project analyzes the relationship between marketing expenditure and generated revenue using Linear Regression based on the Least Squares Estimation method. The objective is to estimate model parameters that best explain how marketing investment influences revenue.

## Problem Statement
Organizations invest large amounts of money in marketing activities such as advertising, promotions, and digital campaigns. However, understanding how marketing expenditure impacts revenue generation is important for making data-driven decisions.

This project builds a regression model to analyze the relationship between marketing spend and revenue using the Least Squares Estimation technique.

## Objective
- Estimate parameters of a regression model using Least Squares Estimation
- Analyze the relationship between marketing spend and revenue
- Predict revenue based on marketing investment
- Visualize the regression relationship using graphs

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Methodology
1. Generate a dataset containing marketing spend and revenue values
2. Apply Linear Regression using Least Squares Estimation
3. Estimate model parameters (intercept and slope)
4. Evaluate model performance using R² score
5. Visualize the relationship between marketing spend and revenue
6. Predict revenue for future marketing investments

## Regression Model
The relationship between marketing spend and revenue is represented using the equation:

Revenue = a + b × Marketing Spend

Where:
- **a** represents the intercept
- **b** represents the slope of the regression line

## Results
The regression model successfully estimates the relationship between marketing expenditure and revenue. Visualization graphs demonstrate that revenue increases with higher marketing spending.
